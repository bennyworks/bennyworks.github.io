### list contains(list, element)

如果给定的列表包含该元素，则返回 `true`。否则，返回 `false`。

**函数声明**

```
list contains(list: list, element: Any): boolean
```

**示例**

```
list contains([1,2,3], 2)
// true
```

### count(list)

返回给定列表的元素数量。

**函数声明**

```
count(list: list): number
```

**示例**

```
count([1,2,3])
// 3
```

### min(list)

返回给定列表的最小值。

**函数声明**

```
min(list: list): Any
```

`list`中的所有元素应具有相同的类型，并且可以比较。

参数`list`可以传递为列表或元素序列。

**示例**

```
min([1,2,3])
// 1

min(1,2,3)
// 1
```

### all(list)

如果给定列表中的任何元素为`false`，则返回`false`。否则，返回`true`。

如果给定列表为空，则返回`true`。

**函数声明**

```
all(list: list<boolean>): boolean
```

参数`list`可以传递为列表或元素序列。

**示例**

```
all([true,false])
// false

all(false,null,true)
// false
```

:::info[提示]

函数`all()`替换了以前的函数`and()`。以前的函数已被弃用，不应再使用。

:::

### any(list)

如果给定列表中的任何元素为`true`，则返回`true`。否则，返回`false`。

如果给定列表为空，则返回`false`。

**函数声明**

```
any(list: list<boolean>): boolean
```

参数`list`可以传递为列表或元素序列。

**示例**

```
any([false,true])
// true

any(false,null,true)
// true
```

:::info[提示]

函数`any()`替换了以前的函数`or()`。以前的函数已被弃用，不应再使用。

:::

### sublist(list, start position)

从`start position`开始返回给定值的部分列表。

**函数声明**

```
sublist(list: list, start position: number): list

```

`start position`的起始位置索引为`1`。最后一个位置的索引是`-1`。

**示例**

```
sublist([1,2,3], 2)
// [2,3]
```

### sublist(list, start position, length)

从`start position`开始返回给定值的部分列表，列表长度不超过`length`。

**函数声明**

```
sublist(list: list, start position: number, length: number): list
```

`start position`的起始位置索引为`1`。最后一个位置的索引是`-1`。

**示例**

```
sublist([1,2,3], 1, 2)
// [1,2]
```

参数`lists`是一系列列表。

### string join(list)

将字符串列表连接成一个字符串。这与 Java 的 [joining](https://docs.oracle.com/en/java/javase/11/docs/api/java.base/java/util/stream/Collectors.html#joining(java.lang.CharSequence,java.lang.CharSequence,java.lang.CharSequence)) 函数类似。

如果列表项是 `null`，则该项将被忽略。如果项既不是字符串也不是 `null`，则该函数返回 `null` 而不是字符串。

**函数声明**

```
string join(list: list<string>): string
```

**示例**

```
string join(["a","b","c"])
// "abc"

string join(["a",null,"c"])
// "ac"

string join([])
// ""
```

### string join(list, delimiter)

将字符串列表连接成一个字符串，并用`delimiter`分隔。这与 Java 的 [joining](https://docs.oracle.com/en/java/javase/11/docs/api/java.base/java/util/stream/Collectors.html#joining(java.lang.CharSequence,java.lang.CharSequence,java.lang.CharSequence)) 函数类似。

如果列表项是 `null`，则该项将被忽略。如果项既不是字符串也不是 `null`，则该函数返回 `null` 而不是字符串。

返回的字符串中每个元素之间都有一个 `delimiter`。

**函数声明**

```
string join(list: list<string>, delimiter: string): string
```

**示例**

```
string join(["a"], "X")
// "a"

string join(["a","b","c"], ", ")
// "a, b, c"
```

### string join(list, delimiter, prefix, suffix)

<Highlight color="#c54302">Camunda Extension</Highlight>

将字符串列表连接成一个字符串。这与 Java 的 [joining](https://docs.oracle.com/en/java/javase/11/docs/api/java.base/java/util/stream/Collectors.html#joining(java.lang.CharSequence,java.lang.CharSequence,java.lang.CharSequence)) 函数类似。

如果列表项是 `null`，则该项将被忽略。如果项既不是字符串也不是 `null`，则该函数返回 `null` 而不是字符串。

返回的字符串以 `prefix` 开头，每个元素之间都有一个 `delimiter`，并以 `suffix` 结尾。

**函数声明**

```
string join(list: list<string>, delimiter: string, prefix: string, suffix: string): string
```

**示例**

```
string join(["a","b","c"], ", ", "[", "]")
// "[a, b, c]"
```
