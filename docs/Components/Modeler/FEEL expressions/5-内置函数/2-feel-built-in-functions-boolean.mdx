### not(negand)

返回给定值的逻辑否定。

**函数声明**


```java
not(negand: boolean): boolean
```

**示例**

```java
not(true)
// false

not(null)
// null
```

### is defined(value)

<Highlight color="#c54302">Camunda Extension</Highlight>


检查给定值是否为 `null`。如果值是 `null` 则函数返回 `false`。否则，函数返回 `true`。

**函数声明**

```java
is defined(value: Any): boolean
```

**示例**

```java
is defined(1)
// true

is defined(null)
// false

is defined(x)
// false - 如果不存在变量 "x"

is defined(x.y)
// false - 如果不存在变量 "x" 或它没有属性 "y"
```

:::warning[重大调整]

这个函数跟以前的版本有较大的差异。如果值是 `null` 则函数返回 `true`。从这一版本开始，如果值是 `null` 则函数返回 `false`。

:::

### get or else(value, default)

<Highlight color="#c54302">Camunda Extension</Highlight>

如果参数值不为 `null` 则返回参数值，否则返回默认值。

**函数声明**

```java
get or else(value: Any, default: Any): Any
```

**示例**

```java
get or else("this", "default")
// "this"

get or else(null, "default")
// "default"

get or else(null, null)
// null
```

### assert(value, condition)

<Highlight color="#c54302">Camunda Extension</Highlight>

验证给定的条件是否满足。如果条件为 `true`，则返回参数值。否则，运算失败并抛出`error`。


**函数声明**

```java
assert(value: Any, condition: Any)
```

**示例**

```java
assert(x, x != null)
// 8 - 如果 x 的值为 8
// error - 如果 x 的值为 null 或不存在

assert(x, x >= 0)
// 8 - 如果 x 的值为 8
// error - 如果 x 小于 0
```

### assert(value, condition, cause)

<Highlight color="#c54302">Camunda Extension</Highlight>

验证给定的条件是否满足。如果条件为 `true`，则返回参数值。否则，运算失败并抛出包含给定文本信息的`error`。

**函数声明**

```java
assert(value: Any, condition: Any, cause: String)
```

**示例**

```java
assert(x, x != null, "'x' 不能为null")
// 8 - 如果 x 的值为 8
// error('x' should not be null) - if x is null or doesn't exist
// error('x' 不能为null) - 如果 x 为 null 或不存在

assert(x, x >= 0, "'x' 须为正数")
// 8 - 如果 x 的值为 8
// error('x' 须为正数) - 如果 x 小于 0
```
