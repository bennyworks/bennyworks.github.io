### substring(string, start position)

返回给定值从 `start position` 开始的子字符串。

**函数声明**

```java
substring(string: string, start position: number): string
```

`start position` 的索引从1开始，最后一位的索引用-1表示。

**示例**

```java
substring("foobar", 3)
// "obar"

substring("foobar", -2)
// "ar"
```

### substring(string, start position, length)

返回给定值从 `start position` 开始的子字符串，长度为 `length`。如果 `length` 大于给定值剩余的字符数，则返回从 `start position` 到值末尾的所有字符。

**函数声明**

```java
substring(string: string, start position: number, length: number): string
```

`start position` 的索引从1开始，最后一位的索引用-1表示。

**示例**

```java
substring("foobar", 3, 3)
// "oba"

substring("foobar", -3, 2)
// "ba"

substring("foobar", 3, 10)
// "obar"

```

### matches(input, pattern)

如果给定的值与正则表达式`pattern`匹配，则返回`true`。否则，返回`false`。

**函数声明**

```java
matches(input: string, pattern: string): boolean
```

`pattern` 是一个正则表达式。


**示例**

```java
matches("foobar", "^fo*bar")
// true
```

### matches(input, pattern, flags)

如果给定的值与正则表达式`pattern`匹配，则返回`true`。否则，返回`false`。

**函数声明**

```java
matches(input: string, pattern: string, flags: string): boolean
```

`pattern` 是一个正则表达式。
`flags` 可以包含以下一个或多个字符：

- `s` (dot-all，点号`.`匹配包括换行符在内的任何单个字符)
- `m` (multi-line，多行模式，^和$匹配每一行的开始和结束)
- `i` (case insensitive，不区分大小写)
- `x` (comments，忽略正则表达式中的空白字符，除非它们是反斜杠转义或位于字符类中)

**示例**

```java
matches("FooBar", "foo", "i")
// true
```

### replace(input, pattern, replacement)

在给定的值中，将所有`pattern`替换为`replacement`，并返回结果字符串。


**函数声明**

```java
replace(input: string, pattern: string, replacement: string): string
```

`pattern` 是一个正则表达式。

使用`$` 和组号来访问匹配组，例如，`$1` 来访问第一组。

**示例**

```java
replace("abcd", "(ab)|(a)", "[1=$1][2=$2]")
// "[1=ab][2=]cd"

replace("0123456789", "(\d{3})(\d{3})(\d{4})", "($1) $2-$3")
// "(012) 345-6789"
```

### replace(input, pattern, replacement, flags)

在给定的值中，将所有`pattern`替换为`replacement`，并返回结果字符串。

**函数声明**

```java
replace(input: string, pattern: string, replacement: string, flags: string): string
```

`pattern` 是一个正则表达式。

使用`$` 和组号来访问匹配组，例如，`$1` 来访问第一组。

`flags` 可以包含以下一个或多个字符：

- `s` (dot-all，点号`.`匹配包括换行符在内的任何单个字符)
- `m` (multi-line，多行模式，^和$匹配每一行的开始和结束)
- `i` (case insensitive，不区分大小写)
- `x` (comments，忽略正则表达式中的空白字符，除非它们是反斜杠转义或位于字符类中)

**示例**

```java
replace("How do you feel?", "Feel", "FEEL", "i")
// "How do you FEEL?"
```

### split(string, delimiter)

根据 `delimiter` 模式将给定的值拆分为子字符串列表，在每个出现的位置进行拆分。

**函数声明**

```java
split(string: string, delimiter: string): list<string>
```

`delimiter` 是一个正则表达式。

**示例**

```java
split("John Doe", "\s" )
// ["John", "Doe"]

split("a;b;c;;", ";")
// ["a", "b", "c", "", ""]
```

### extract(string, pattern)

<Highlight color="#c54302">Camunda Extension</Highlight>

返回给定字符串中所有匹配项的模式。如果模式不匹配，则返回空列表。

**函数声明**

```java
extract(string: string, pattern: string): list<string>
```

`pattern` 是一个正则表达式。

**示例**

```java
extract("references are 1234, 1256, 1378", "12[0-9]*")
// ["1234","1256"]
```
