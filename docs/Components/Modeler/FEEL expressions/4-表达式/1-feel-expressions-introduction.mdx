### 注释
表达式可以包含注释来解释它并提供更多的上下文。这可以通过使用 Java 风格的注释来完成：// 到行尾，或者 /*.... */ 用于块。

```java
// 返回最后一个项目
[1,2,3,4][-1]

/* 返回最后一个项目 */
[1,2,3,4][-1]

/*
 * 返回最后一个项目
 */
[1,2,3,4][-1]
```

### 括号
括号（..）可以在表达式中用于分隔表达式的不同部分或影响运算符的优先级。

```java
(5 - 3) * (4 / 2)

x < 5 and (y > 10 or z > 20)

if (5 < 10) then "low" else "high"
```
