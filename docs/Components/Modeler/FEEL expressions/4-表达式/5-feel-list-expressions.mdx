### 字面量 

创建一个包含给定元素的新列表。元素可以是任何类型。

```java
[1,2,3,4]
```

一个列表值可以嵌入其他列表值。

```java
[[1,2], [3,4], [5,6]]
```

### 获取元素

```java
a[i]
```

访问列表`a` 中索引为`i`的元素。索引从`1`开始。

如果索引超出列表的范围，则返回`null`。

```java
[1,2,3,4][1]
// 1

[1,2,3,4][2]
// 2

[1,2,3,4][4]
// 4

[1,2,3,4][5]
// null

[1,2,3,4][0]
// null
```

如果索引为负数，则从列表末尾开始计数。列表的最后一个元素在索引 `-1` 处。

```java
[1,2,3,4][-1]
// 4

[1,2,3,4][-2]
// 3

[1,2,3,4][-5]
// null
```

:::warning[注意]

列表的索引从 `1` 开始。在其他语言中，索引从 `0` 开始。

:::

### 过滤

```java
a[c]
```

通过条件 `c` 过滤列表 `a`。表达式的结果是一个包含所有条件 `c` 评估为 `true` 的元素的列表。其他元素被排除。

在过滤过程中，当前元素被赋值给变量 `item`。

```java
[1,2,3,4][item > 2]
// [3,4]

[1,2,3,4][item > 10]
// []

[1,2,3,4][even(item)]
// [2,4]
```

### Some运算

```java
some a in b satisfies c

```

遍历列表 `b`，并对列表中的每个元素进行条件 `c`运算。当前元素被赋值给变量 `a`。

如果 `c` 对 `b` 的一个或多个元素求值为 `true`，则返回 `true`。否则，返回 `false`。

```java
some x in [1,2,3] satisfies x > 2
// true

some x in [1,2,3] satisfies x > 5
// false

some x in [1,2,3] satisfies even(x)
// true

some x in [1,2], y in [2,3] satisfies x < y
// true
```

### Every运算

遍历列表 `b`，并对列表中的每个元素进行条件 `c`运算。当前元素被赋值给变量 `a`。

如果 `c` 对所有 `b` 的元素求值为 `true`，则返回 `true`。否则，返回 `false`。

```java
every x in [1,2,3] satisfies x >= 1
// true

every x in [1,2,3] satisfies x >= 2
// false

every x in [1,2,3] satisfies even(x)
// false

every x in [1,2], y in [2,3] satisfies x < y
// false
```
