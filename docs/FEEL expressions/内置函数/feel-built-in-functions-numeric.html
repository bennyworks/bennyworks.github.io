Numeric 函数

decimal(n, scale)​

根据给定的精度四舍五入。

函数声明

decimal(n: number, scale: number): number

示例

decimal(1/3, 2)
// .33

decimal(1.5, 0)
// 2

floor(n)​

向下取整。

函数声明

floor(n: number): number

示例

floor(1.5)
// 1

floor(-1.5)
// -2

floor(n, scale)​

根据给定的精度向负无穷的方向舍入。

函数声明

floor(n: number, scale: number): number

示例

floor(-1.56, 1)
// -1.6

ceiling(n)​

向上取整。

函数声明

ceiling(n: number): number

示例

ceiling(1.5)
// 2

ceiling(-1.5)
// -1

ceiling(n, scale)​

根据给定的精度向正无穷方向舍入。

函数声明

ceiling(n: number, scale: number): number

示例

ceiling(-1.56, 1)
// -1.5

round up(n, scale)​

根据给定的精度对数字向上取整。

函数声明

round up(n: number, scale: number): number

示例

round up(5.5)
// 6

round up(-5.5)
// -6

round up(1.121, 2)
// 1.13

round up(-1.126, 2)
// -1.13

round down(n, scale)​

根据给定的精度对数值向下取整。

函数声明

round down(n: number, scale: number): number

示例

round down(5.5, 0)
// 5

round down (-5.5, 0)
// -5

round down (1.121, 2)
// 1.12

round down (-1.126, 2)
// -1.12

round half up(n, scale)​

根据给定的精度，以0.5作为边界向上舍入。

函数声明

round half up(n: number, scale: number): number

示例

round half up(5.5, 0)
// 6

round half up(-5.5, 0)
// -6

round half up(1.121, 2)
// 1.12

round half up(-1.126, 2)
// -1.13

round half down(n, scale)​

根据给定的精度，以0.5作为边界向下舍入。

函数声明

round half down(n: number, scale: number): number

示例

round half down (5.5, 0)
// 5

round half down (-5.5, 0)
// -5

round half down (1.121, 2)
// 1.12

round half down (-1.126, 2)
// -1.13

abs(number)​

返回绝对值。

函数声明

abs(number: number): number

示例

abs(10)
// 10

abs(-10)
// 10

modulo(dividend, divisor)​

返回被除数除以除数的余数。

函数声明

modulo(dividend: number, divisor: number): number

示例

modulo(12, 5)
// 2

sqrt(number)​

返回给定值的平方根。

函数声明

sqrt(number: number): number

示例

sqrt(16)
// 4

log(number)​

返回给定值的自然对数（底数为 e）。

函数声明

log(number: number): number

示例

log(10)
// 2.302585092994046

exp(number)​

返回给定数字的欧拉数 e 的幂。

函数声明

exp(number: number): number

示例

exp(5)
// 148.4131591025766

odd(number)​

如果给定值为奇数，则返回true。否则，返回false。

函数声明

odd(number: number): boolean

示例

odd(5)
// true

odd(2)
// false

even(number)​

如果给定值为偶数，则返回true。否则，返回false。

函数声明

even(number: number): boolean

示例

even(5)
// false

even(2)
// true

random number()​

Camunda Extension

返回一个在0和1之间的随机数。

函数声明

random number(): number

示例

random number()
// 0.9701618132579795
