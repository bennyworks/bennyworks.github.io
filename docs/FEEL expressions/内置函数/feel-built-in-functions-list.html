List 函数

list contains(list, element)​

如果给定的列表包含该元素，则返回 true。否则，返回 false。

函数声明

list contains(list: list, element: Any): boolean

示例

list contains([1,2,3], 2)
// true

count(list)​

返回给定列表的元素数量。

函数声明

count(list: list): number

示例

count([1,2,3])
// 3

min(list)​

返回给定列表的最小值。

函数声明

min(list: list): Any

list中的所有元素应具有相同的类型，并且可以比较。

参数list可以传递为列表或元素序列。

示例

min([1,2,3])
// 1

min(1,2,3)
// 1

max(list)​

返回给定列表的最大值。

函数声明

max(list: list): Any

list中的所有元素应具有相同的类型，并且可以比较。

参数list可以传递为列表或元素序列。

示例

max([1,2,3])
// 3

max(1,2,3)
// 3

sum(list)​

返回给定数字列表的总和。

函数声明

sum(list: list<number>): number

参数list可以传递为列表或元素序列。

示例

sum([1,2,3])
// 6

sum(1,2,3)
// 6

product(list)​

返回给定数字列表的乘积。

函数声明

product(list: list<number>): number

参数list可以传递为列表或元素序列。

示例

product([2, 3, 4])
// 24

product(2, 3, 4)
// 24

mean(list)​

返回给定数字列表的算术平均值（即平均值）。

函数声明

mean(list: list<number>): number

参数list可以传递为列表或元素序列。

示例

mean([1,2,3])
// 2

mean(1,2,3)
// 2

median(list)​

返回给定数字列表的中位数元素。

函数声明

median(list: list<number>): number

参数list可以传递为列表或元素序列。

示例

median(8, 2, 5, 3, 4)
// 4

median([6, 1, 2, 3])
// 2.5

stddev(list)​

返回给定数字列表的标准差。

函数声明

stddev(list: list<number>): number

参数list可以传递为列表或元素序列。

示例

stddev(2, 4, 7, 5)
// 2.0816659994661326

stddev([2, 4, 7, 5])
// 2.0816659994661326

mode(list)​

返回给定数字列表的众数。

函数声明

mode(list: list<number>): number

参数list可以传递为列表或元素序列。

示例

mode(6, 3, 9, 6, 6)
// [6]

mode([6, 1, 9, 6, 1])
// [1, 6]

all(list)​

如果给定列表中的任何元素为false，则返回false。否则，返回true。

如果给定列表为空，则返回true。

函数声明

all(list: list<boolean>): boolean

参数list可以传递为列表或元素序列。

示例

all([true,false])
// false

all(false,null,true)
// false
提示

函数all()替换了以前的函数and()。以前的函数已被弃用，不应再使用。

any(list)​

如果给定列表中的任何元素为true，则返回true。否则，返回false。

如果给定列表为空，则返回false。

函数声明

any(list: list<boolean>): boolean

参数list可以传递为列表或元素序列。

示例

any([false,true])
// true

any(false,null,true)
// true
提示

函数any()替换了以前的函数or()。以前的函数已被弃用，不应再使用。

sublist(list, start position)​

从start position开始返回给定值的部分列表。

函数声明

sublist(list: list, start position: number): list

start position的起始位置索引为1。最后一个位置的索引是-1。

示例

sublist([1,2,3], 2)
// [2,3]

sublist(list, start position, length)​

从start position开始返回给定值的部分列表，列表长度不超过length。

函数声明

sublist(list: list, start position: number, length: number): list

start position的起始位置索引为1。最后一个位置的索引是-1。

示例

sublist([1,2,3], 1, 2)
// [1,2]

append(list, items)​

将所有items附加到给定列表中并返回。

函数声明

append(list: list, items: Any): list

items参数可以是一个单独的元素或一系列元素。

示例

append([1], 2, 3)
// [1,2,3]

concatenate(lists)​

将所有给定列表的元素组合成一个列表并返回。

函数声明

concatenate(lists: list): list

参数lists是一系列列表。

示例

concatenate([1,2],[3])
// [1,2,3]

concatenate([1],[2],[3])
// [1,2,3]

insert before(list, position, newItem)​

将newItem插入到position并返回给定列表。

函数声明

insert before(list: list, position: number, newItem: Any): list

position的起始位置索引为1。最后一个位置的索引是-1。

示例

insert before([1,3],1,2)
// [2,1,3]

remove(list, position)​

将position处的元素从给定列表中移除并返回。

函数声明

remove(list: list, position: number): list

position的起始位置索引为1。最后一个位置的索引是-1。

示例

remove([1,2,3], 2)
// [1,3]

reverse(list)​

返回给定列表的反向顺序。

函数声明

reverse(list: list): list

示例

reverse([1,2,3])
// [3,2,1]

index of(list, match)​

将给定列表中匹配match的元素的索引按升序列表返回。

函数声明

index of(list: list, match: Any): list<number>

示例

index of([1,2,3,2],2)
// [2,4]

union(lists)​

将给定的多个列表的所有元素组合成一个列表并返回，不包含重复项。

函数声明

union(lists: list): list

参数lists是一系列列表。

示例

union([1,2],[2,3])
// [1,2,3]

distinct values(list)​

去重

函数声明

distinct values(list: list): list

示例

distinct values([1,2,3,2,1])
// [1,2,3]

duplicate values(list)​

Camunda Extension

返回给定列表的所有重复值。

函数声明

duplicate values(list: list): list

示例

duplicate values([1,2,3,2,1])
// [1,2]

flatten(list)​

将给定列表的所有元素组合成一个列表并返回，不包含嵌套列表。

函数声明

flatten(list: list): list

示例

flatten([[1,2],[[3]], 4])
// [1,2,3,4]

sort(list, precedes)​

按precedes函数对给定列表进行排序。

函数声明

sort(list: list, precedes: function<(Any, Any) -> boolean>): list

示例

sort(list: [3,1,4,5,2], precedes: function(x,y) x < y)
// [1,2,3,4,5]

string join(list)​

将字符串列表连接成一个字符串。这与 Java 的 joining 函数类似。

如果列表项是 null，则该项将被忽略。如果项既不是字符串也不是 null，则该函数返回 null 而不是字符串。

函数声明

string join(list: list<string>): string

示例

string join(["a","b","c"])
// "abc"

string join(["a",null,"c"])
// "ac"

string join([])
// ""

string join(list, delimiter)​

将字符串列表连接成一个字符串，并用delimiter分隔。这与 Java 的 joining 函数类似。

如果列表项是 null，则该项将被忽略。如果项既不是字符串也不是 null，则该函数返回 null 而不是字符串。

返回的字符串中每个元素之间都有一个 delimiter。

函数声明

string join(list: list<string>, delimiter: string): string

示例

string join(["a"], "X")
// "a"

string join(["a","b","c"], ", ")
// "a, b, c"

string join(list, delimiter, prefix, suffix)​

Camunda Extension

将字符串列表连接成一个字符串。这与 Java 的 joining 函数类似。

如果列表项是 null，则该项将被忽略。如果项既不是字符串也不是 null，则该函数返回 null 而不是字符串。

返回的字符串以 prefix 开头，每个元素之间都有一个 delimiter，并以 suffix 结尾。

函数声明

string join(list: list<string>, delimiter: string, prefix: string, suffix: string): string

示例

string join(["a","b","c"], ", ", "[", "]")
// "[a, b, c]"
