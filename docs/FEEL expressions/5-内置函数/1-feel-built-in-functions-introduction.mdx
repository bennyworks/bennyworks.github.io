title: 介绍

FEEL 包含许多内置函数。这些函数可以在[表达式](../4-表达式/1-feel-expressions-introduction.mdx)和[一元测试](../3-feel-unary-tests.mdx)中调用。

```java
contains("me@camunda.com", ".com")
// 按参数位置传参调用函数

contains(string: "me@camunda.com", match: ".de")
// 按参数名称传参调用函数
```

更多关于函数的信息，请参阅[这里](../4-表达式/10-feel-functions.mdx#调用)。

基于主要操作数据类型，可以将函数分为以下几类：

- [Boolean](./2-feel-built-in-functions-boolean.mdx)
- [String](./3-feel-built-in-functions-string.mdx)
- [Numeric](./4-feel-built-in-functions-numeric.mdx)
- [List](./5-feel-built-in-functions-list.mdx)
- [Context](./6-feel-built-in-functions-context.mdx)
- [Temporal](./7-feel-built-in-functions-temporal.mdx)
- [Range](./8-feel-built-in-functions-range.mdx)

另外，还提供了一些[转换函数](./9-feel-built-in-functions-conversion.mdx)用于构造新数据类型（工厂函数）。
