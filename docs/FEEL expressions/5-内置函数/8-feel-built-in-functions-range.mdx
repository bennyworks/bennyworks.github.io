title: Range 函数
---

Range函数是一组函数，用于建立单个标量值和这些值的范围之间的关系。所有函数都接受两个参数，并在关系成立时返回`true`，否则返回`false`。

一个标量值必须是以下类型之一：

- number
- date
- time
- date-time
- days-time-duration
- years-months-duration

![Range函数概览](/docs/assets/feel-built-in-functions-range-overview.png)

### before(point1, point2)

**函数声明**

```
before(point1: Any, point2: Any): boolean
```

**示例**

```
before(1, 10)
// true

before(10, 1)
// false
```

### before(range, point)

**函数声明**

```
before(range: range, point: Any): boolean
```

**示例**

```
before([1..5], 10)
// true
```

### before(point, range)

**函数声明**

```
before(point: Any, range: range): boolean
```

**示例**

```
before(1, [2..5])
// true
```

### before(range1, range2)

**函数声明**

```
before(range1: range, range2: range): boolean
```

**示例**

```
before([1..5], [6..10])
// true

before([1..5),[5..10])
// true
```
